Algorithm

1. Pick a starting value.
2. Pick number of iterations.
3. Calculate the new point while xnew - xold > precision.

         xnew := xold - eps * df/dxold

Epsilon is calculating as:
    z(eps) = x - eps * df/dxold
Find eps point where f'(z(eps)) = 0
